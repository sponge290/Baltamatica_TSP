# 文档一：天气感知型TSP项目技术方案（替换北太天元版）

核心修改说明：**彻底移除北太天元本地计算与本地代理架构**，替换为「Supabase Edge Functions (Deno) 全Serverless后端计算」，彻底解决连接问题；保留原项目所有其他设计（Vercel前端、Supabase数据库、D3.js可视化、预设测试用例），改动最小、落地最快，完美适配Trae Builder&MCP开发模式。

---

## 一、项目核心架构（重构版）

### 1.1 架构总览

采用**全Serverless三层架构**，零本地依赖、零连接问题、零服务器运维：

1. **公网前端层**：Vercel静态托管，保持原设计不变，负责界面交互、参数配置、D3.js可视化

2. **数据持久层**：Supabase PostgreSQL，保持原表结构不变，负责基础数据、测试用例、求解结果存储

3. **Serverless计算层**：Supabase Edge Functions (Deno)，**替换原北太天元本地计算**，负责TSP算法执行、天气影响量化、约束条件校验

### 1.2 核心优势（解决原方案所有痛点）

1. **彻底解决连接问题**：无本地代理、无本地软件依赖，用户打开网站即可使用，零配置门槛

2. **全Serverless零运维**：无需购买服务器、无需部署本地环境，Vercel+Supabase免费额度覆盖个人使用

3. **完美适配Trae MCP**：Edge Functions基于Deno/TypeScript，可直接用Trae MCP辅助开发、调试、部署

4. **保留所有原有功能**：数据库设计、前端界面、D3.js可视化、预设测试用例完全复用，仅替换计算层

5. **用户体验大幅提升**：无需下载脚本、无需运行本地软件，打开网站即可计算，分享链接无需对方安装任何软件

### 1.3 全链路数据流向

1. 前端从Supabase拉取预设测试用例与基础数据

2. 用户选择用例、配置算法参数，点击「开始计算」

3. 前端通过HTTP请求调用Supabase Edge Function

4. Edge Function接收参数，执行TSP算法、天气影响量化、约束校验

5. Edge Function返回标准JSON结果，同时异步写入Supabase数据库

6. 前端接收结果，通过D3.js实现实时可视化

7. 用户可查看结果、对比性能、生成分享链接

### 4.3 入口函数设计

```TypeScript

// index.ts
import { serve } from "https://deno.land/std@0.192.0/http/server.ts";
import { createClient } from "https://esm.sh/@supabase/supabase-js@2";
import { solveDP } from "./algorithms/dp.ts";
import { solveAStar } from "./algorithms/astar.ts";
import { solveGA } from "./algorithms/ga.ts";
import { validateRequest } from "./utils/constraints.ts";
import type { SolveTSPRequest, TSPSolution } from "./utils/types.ts";

const supabase = createClient(
  Deno.env.get("SUPABASE_URL")!,
  Deno.env.get("SUPABASE_SERVICE_ROLE_KEY")!
);

serve(async (req) => {
  // CORS配置
  if (req.method === "OPTIONS") {
    return new Response(null, {
      headers: {
        "Access-Control-Allow-Origin": "*",
        "Access-Control-Allow-Methods": "POST, OPTIONS",
        "Access-Control-Allow-Headers": "Content-Type",
      },
    });
  }

  try {
    const request: SolveTSPRequest = await req.json();
    
    // 1. 参数校验
    const validation = validateRequest(request);
    if (!validation.valid) {
      return new Response(JSON.stringify({ code: 400, msg: validation.error }), {
        status: 400,
        headers: { "Content-Type": "application/json", "Access-Control-Allow-Origin": "*" },
      });
    }

    // 2. 执行对应算法
    const startTime = Date.now();
    let solution: TSPSolution;
    
    switch (request.algorithm) {
      case "DP":
        solution = await solveDP(request);
        break;
      case "A*":
        solution = await solveAStar(request);
        break;
      case "GA":
        solution = await solveGA(request);
        break;
      default:
        throw new Error("不支持的算法类型");
    }
    
    solution.exec_time = Date.now() - startTime;

    // 3. 异步写入数据库（不阻塞返回）
    (async () => {
      try {
        // 插入路径解
        const { data: solData, error: solError } = await supabase
          .from("route_solutions")
          .insert([{
            case_id: request.case_id,
            algorithm: request.algorithm,
            total_cost: solution.total_cost,
            total_time: solution.total_time,
            reliability: solution.reliability,
            exec_time: solution.exec_time,
            route_sequence: solution.best_path,
            is_public: false
          }])
          .select()
          .single();

        if (solError) throw solError;

        // 插入路径节点
        const { error: nodesError } = await supabase
          .from("route_nodes")
          .insert(solution.nodes.map((node, idx) => ({
            solution_id: solData.solution_id,
            city_id: node.city_id,
            visit_order: idx + 1,
            arrival_time: node.arrival_time,
            departure_time: node.departure_time,
            weather_condition: node.weather_condition
          })));

        if (nodesError) throw nodesError;
      } catch (dbError) {
        console.error("数据库写入失败:", dbError);
      }
    })();

    // 4. 返回结果给前端
    return new Response(JSON.stringify({ code: 200, data: solution }), {
      headers: { "Content-Type": "application/json", "Access-Control-Allow-Origin": "*" },
    });

  } catch (error) {
    console.error("执行失败:", error);
    return new Response(JSON.stringify({ 
      code: 500, 
      msg: "执行失败", 
      error: error instanceof Error ? error.message : "未知错误" 
    }), {
      status: 500,
      headers: { "Content-Type": "application/json", "Access-Control-Allow-Origin": "*" },
    });
  }
});
```

### 4.4 算法实现核心逻辑（TypeScript）

#### 4.4.1 动态规划算法（DP）

```TypeScript

// algorithms/dp.ts
import type { SolveTSPRequest, TSPSolution } from "../utils/types.ts";
import { calculateWeatherAwareTime, isTimeWindowValid, calculatePathMetrics } from "../utils/weather.ts";

export async function solveDP(request: SolveTSPRequest): Promise<TSPSolution> {
  const { cities, time_windows, weather_data, road_segments } = request;
  const n = cities.length;
  
  if (n > 15) {
    throw new Error("动态规划仅支持15个城市以内的小规模问题");
  }

  const INF = 1e18;
  const dp: number[][] = Array(2 ** n).fill(0).map(() => Array(n).fill(INF));
  const prev: number[][] = Array(2 ** n).fill(0).map(() => Array(n).fill(-1));
  
  dp[1][0] = 0; // 起点为第0个城市

  // 状态转移
  for (let mask = 1; mask < 2 ** n; mask++) {
    for (let u = 0; u < n; u++) {
      if ((mask & (1 << u)) !== 0 && dp[mask][u] < INF) {
        for (let v = 0; v < n; v++) {
          if ((mask & (1 << v)) === 0) {
            const currentTime = dp[mask][u];
            const { travelTime, cost, reliability } = calculateWeatherAwareTime(
              u, v, currentTime, cities, road_segments, weather_data
            );
            const arrivalTime = currentTime + travelTime;
            
            if (isTimeWindowValid(v, arrivalTime, time_windows)) {
              const newMask = mask | (1 << v);
              if (dp[newMask][v] > dp[mask][u] + cost) {
                dp[newMask][v] = dp[mask][u] + cost;
                prev[newMask][v] = u;
              }
            }
          }
        }
      }
    }
  }

  // 回溯最优路径
  let fullMask = (1 << n) - 1;
  let minCost = INF;
  let bestIdx = 0;
  
  for (let v = 0; v < n; v++) {
    const returnCost = calculateWeatherAwareTime(v, 0, dp[fullMask][v], cities, road_segments, weather_data).cost;
    if (dp[fullMask][v] + returnCost < minCost) {
      minCost = dp[fullMask][v] + returnCost;
      bestIdx = v;
    }
  }

  // 重构路径
  let bestPath: number[] = [];
  let currentMask = fullMask;
  let current = bestIdx;
  
  while (current !== -1) {
    bestPath.unshift(current);
    const next = prev[currentMask][current];
    currentMask = currentMask ^ (1 << current);
    current = next;
  }
  bestPath.push(0); // 返回起点

  // 计算全量指标
  const { totalTime, reliability, nodes } = calculatePathMetrics(
    bestPath, cities, time_windows, weather_data, road_segments
  );

  return {
    best_path: bestPath,
    total_cost: minCost,
    total_time: totalTime,
    reliability,
    exec_time: 0,
    nodes,
    process_data: { dp_table: dp } // 可视化用
  };
}
```

#### 4.4.2 A*算法与遗传算法

- A*算法：基于TypeScript优先队列实现，启发式函数采用最小生成树下界

- 遗传算法：基于TypeScript数组操作实现，支持轮盘赌选择、顺序交叉、交换变异

- 完整实现逻辑参考原北太天元算法框架，仅语法替换为TypeScript
